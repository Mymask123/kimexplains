Personal Website

In my free time I created a personal website using HTML, CSS, and JavaScript. The website serves as a portfolio to showcase my projects, achievements, and blog posts. I designed the layout to be clean and modern, with easy navigation between different sections. I also implemented a dark mode feature using JavaScript, allowing users to switch between light and dark themes for better readability. This was a great way for me to practice my web development skills and create an online presence. I also wanted to make a place for me to share my thoughts and experiences through blog posts.
